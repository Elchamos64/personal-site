About the website

This website is designed for BYU-Idaho students. It’s a website that is free and it can hopefully be a great tool when trying to find an apartment that fits your needs.

Q&A

Is it for married students?

-No, this website is designed for single students, it has two categories: private rooms and shared rooms.

Is it only approved housing?

-Yes, only approved housing.

How often is the information updated?

-At least once a semester.

Do I need to create an account to use it?

-No, you don’t have to.
